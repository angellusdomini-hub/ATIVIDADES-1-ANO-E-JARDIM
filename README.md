# Atividades de Teclado - Jardim III e 1º Ano

Pasta pronta para GitHub Pages.

## Arquivos
- `index.html`: página inicial para escolha da turma.
- `jardim3.html`: atividade Caça-Letras no Teclado para Jardim III.
- `1ano.html`: Laboratório do Teclado para 1º Ano.

## Como publicar no GitHub Pages
1. Crie um repositório no GitHub.
2. Envie todos os arquivos desta pasta.
3. Vá em Settings > Pages.
4. Em Branch, selecione `main` e `/root`.
5. Salve e aguarde o link ser gerado.
